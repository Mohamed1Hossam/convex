2.3 MATHEMATICAL JUSTIFICATION
----------------------------------------------------------------------
The objective function is composed of two parts:

1) Quadratic risk term: x^T Σ x
   • A quadratic form x^T Σ x is convex if and only if Σ is positive semi-definite (PSD).
   • From the eigenvalue analysis, the minimum eigenvalue of Σ is non-negative, confirming PSD.
   • Therefore, the risk term is convex.

2) Linear return term: -λ μ^T x
   • Linear (affine) functions are both convex and concave.
   • Minimizing a convex function (Risk) plus an affine function (Negative Return) is a convex problem.

The constraints are all linear, which defines a convex feasible set.
Thus, the entire optimization problem is convex.
